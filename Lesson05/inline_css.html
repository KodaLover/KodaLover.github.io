Font Families

Font families include serif, sans-serif, and monospace. The differences between each font family is appearance. Serif fonts have small flags on each letter, while sans-serif fonts do not. Monospace fonts however, have letters that are an equal amount of space for each letter, no matter the letter's shape.

Color Methods

There are three different ways to change color using CSS: RGB, hexadecimal colors, or color names. RGB colors uses red, green, and blue values to create a color, while hexadecimal colors are represented by codes that include letters, numbers, or a combination of both. Lastly, color names are the simplest form of selecting a color, but the number of available colors is small compared to the other methods.
